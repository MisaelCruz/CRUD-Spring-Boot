# CRUD Spring Boot - Control de Estudiantes

Una aplicación backend desarrollada con Spring Boot para registrar, editar, consultar y eliminar alumnos en una base de datos.

### Tecnologías utilizadas:
* **Java** 
* **Spring Boot**
* **Thymeleaf** (Motor de plantillas para la interfaz web)
* **Gradle** (Gestor de dependencias)
* **MySQL** (Base de datos)
